This program utilises a ROS2 node to extract data from laser scan to determine if a 30cm cylinder is within detecting range of the robot.

Program functionality

This program subscribes to the laserscan topic and reads in data points, organising them into clusters and then determining if those clusters are 30cm wide. If they are then the program publishes a marker to rviz on that cylinders location.

Using the node

To run this program multiple linux terminals must be used.

Terminal 1: export TURTLEBOT3_MODEL=waffle_pi ros2 launch turtlebot3_gazebo turtlebot3_parkworld.launch.py

Terminal 2: export TURTLEBOT3_MODEL=waffle_pi ros2 launch turtlebot3_navigation2 \navigation2.launch.py \ use_sim_time:=true map:=$HOME/smallparkmap.yaml

Termial 3: ros2 run cylinder_detector cylinder_detector

RVIZ: Add marker and subscribe to cylinder_marker Robot will spawn facing "up" on the map in the top right hand corner. Using NAV2GOAL to move robot around or use TELEOP.

Gazebo: Spawn in a cylinder at with a radius of 0.15m and place in the workspace were required.
